# 敲きことはじめへようこそ

## これはなに

弊団体の活動ではよく「敲き」が行われます．
これは成果物に対するアドバイスやフィードバックを行い，より良い成果物へと繋げるためのものです．
そこで敲きについてそのやり方やノウハウをまとめてシェアするために作ったものが「敲きことはじめ」です．
